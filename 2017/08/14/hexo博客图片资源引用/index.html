hexo博客图片资源引用

发表于 2017-08-14 | 分类于 总结

简介

我们在编写博客时通常会遇到图片资源的问题，Windows下的Macdown在引用图片资源时会自动上传到网上，实际上引用的就是网络图片资源了，但是Mac版本的就需要依赖一些插件，或者可以直接引用网络资源，比较好的方法是将图片上传到七牛上，然后在引用对应的网络图片，但是当有大量图片的时候就显得比较麻烦了，所以本文着重介绍一种简单的方法实现图片资源引用，即 CodeFalling/hexo-asset-image插件来加载本地图片

使用方法：

1、首先确认_config.yml 中有 post_asset_folder:true。

Hexo 提供了一种更方便管理 Asset 的设定：post_asset_folder
当您设置post_asset_folder为true参数后，在建立文件时，Hexo会自动建立一个与文章同名的文件夹，您可以把与该文章相关的所有资源都放到那个文件夹，如此一来，您便可以更方便的使用资源。

2、在hexo的目录下执行 npm install https://github.com/CodeFalling/hexo-asset-image –save（需要等待一段时间）。

3、完成安装后用hexo新建文章的时候会发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面。结构如下：

4、把需要引用的图片放到对应的文件夹中，使用MacDown 引用图片的语法即可引用，其中[]里面不写文字则没有图片标题，具体如下：

![image_quote_01](hexo博客图片资源引用/image_quote_01.png)

5、正常上传已经编辑好的带图片的博客，查看一下是否成功。
